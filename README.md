
## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/AlmirGok/TestFrontGOK.git
```

Entre no diretório do projeto

```bash
  cd TestFrontGOK
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```
